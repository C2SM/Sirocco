TESTABLE BEHAVIORS
================================================================================

1. DYNAMIC LEVELS
   - fast_3 completes while slow_1 still running
   - Levels recomputed after each task completion
   - No static level blocking

2. CROSS-DEPENDENCIES
   - medium_2 waits for fast_2 (not medium_1's level)
   - slow_2 waits for medium_2 (not slow_1's level)
   - Proper synchronization at cross-points

3. PRE-SUBMISSION (window_size)
   - window_size=0: No pre-submission, sequential
   - window_size=1: Submit 1 level ahead
   - window_size=2: Submit 2 levels ahead

4. CONVERGENCE
   - finalize waits for ALL branches
   - No premature completion
   - Proper merge point behavior

5. MULTI-CYCLE
   - 3 iterations test cyclic patterns
   - Inter-cycle dependencies via prepare_next
   - Validates cycle lag handling
